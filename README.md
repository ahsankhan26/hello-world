# hello-world
A program that prints Hello World to the console trust me


<h3>Usage</h3>


```ruby
# Clone this repository
$ git clone https://github.com/ahsankhan26/hello-world.git

# Go to the repository folder
$ cd hello-world

# Compile with g++
$ g++ helloWorld.cpp -o helloWorld.exe

# Execute file
$ ./helloWorld.exe
```
